# Chicago Ridership

**Required Package Manager**

* conda (https://docs.anaconda.com/anaconda/install/)

**Installation**

Install conda environment
```
conda env create -f conda_setup.yml
```
Activate conda environment
```
conda activate vianai
```
Run jupyter lab
```
jupyter lab
```
